# CacaPalavrasIA

## English
.NET MAUI app for generating and playing word search puzzles with AI-assisted word selection and local fallback behavior.

### Highlights
- Automatic puzzle generation
- Interactive word search board
- Progress and answer tracking
- Multi-platform .NET MAUI structure

### Project structure
- `Services`: puzzle and word generation logic
- `Models`: puzzle and placement models
- `Views`: reusable game views
- `Platforms`: platform-specific configuration

### Run locally
1. Open `CacaPalavrasIA/CacaPalavrasIA.sln` in Visual Studio 2022.
2. Install the .NET MAUI workload.
3. Restore packages and run on the target platform.

## Português
App em .NET MAUI para gerar e jogar caça-palavras com seleção de palavras apoiada por IA e fallback local.

### Destaques
- Geração automática do puzzle
- Tabuleiro interativo de caça-palavras
- Acompanhamento de progresso e respostas
- Estrutura multiplataforma em .NET MAUI

### Estrutura do projeto
- `Services`: lógica de geração de palavras e do puzzle
- `Models`: modelos do tabuleiro e das posições
- `Views`: views reutilizáveis do jogo
- `Platforms`: configuração específica por plataforma

### Executando localmente
1. Abra `CacaPalavrasIA/CacaPalavrasIA.sln` no Visual Studio 2022.
2. Instale o workload do .NET MAUI.
3. Restaure os pacotes e execute na plataforma desejada.
